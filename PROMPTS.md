Below are the AI prompts used during development.

## Prompt 1
Create a Cloudflare Agents SDK project structure for an AI interview chat app using Workers AI, Durable Objects, a minimal web frontend, and a workflow for generating interview summaries.

## Prompt 2
Generate a TypeScript Cloudflare Worker with:
- a POST /api/chat endpoint
- a POST /api/end endpoint
- an Agent class that stores session state
- Workers AI inference using a Llama 3.3 model
- safe JSON request validation

## Prompt 3
Write a simple HTML, CSS, and vanilla JavaScript chat frontend that can:
- set a role
- send user messages
- render assistant messages
- request an interview summary when the session ends

## Prompt 4
Create README documentation for a Cloudflare AI app submission repository. Include setup, local run steps, architecture, and assignment fit.

## Prompt 5
Improve the interview system prompt so the assistant acts like a professional interviewer, asks concise follow-up questions, and gives clear final feedback without hallucinating user performance outside the chat.
